🔑 Transfer Admin Access

To grant admin access to a new owner, run this SQL once in Supabase Dashboard → SQL Editor:

-- Step 1: Register on the site with the new email
-- Step 2: Run this query:

UPDATE user_data
SET is_admin = true
WHERE data->>'email' = '[email protected]';

-- Step 3: Visit /admin.html and log in

⚠️ Also update ADMIN_EMAILS_FALLBACK in admin.html with the new email.
